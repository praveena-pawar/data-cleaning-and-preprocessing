# Data Cleaning and Preprocessing using Pandas

## Overview
This project focuses on cleaning a dataset using Pandas. The raw data contained missing values and duplicate records, which were handled to make the dataset ready for analysis.

---

## Steps Performed
- Removed duplicate rows  
- Handled missing values using mean (for numerical columns)  
- Replaced missing categorical values with "Unknown"  
- Verified cleaned data using `.isnull()` and `.describe()`  

---

##  Result
- No missing values  
- No duplicate records  
- Clean and structured dataset  

---

## Tools Used
- Python  
- Pandas  
- Jupyter Notebook
